For Relaxing press those key

Press: ENTER key for Rain.

Press: SPASE Key for Piano.

Press: SHIFT key for Windy.

Press: TAB key for Birds.

If you want to pause music press Pause button with you mouse.

You also can play with Arrow keys.

Press: ARROWUP Key for Dropwater

Press: ARROWLEFT key for Car horn 1

Press: ARROWRIGT key for Car horn 2

Press: ARROWDOWN key for Drum
